# SSM Framework Demo

------

**S**SM_Test is an web application based on SSM framework. Developed by **MDipper** & **bajdcc**.

#### Features

1. Role Management.
2. CURD.
3. Authority management, involving GET/POST API.
4. Logging.
5. Async refreshment using ajax, i.e., pjax.

#### Using

- SSM Framework
- Json, pjax & Ajax
- jQuery & jQuery.validate
- Eclipse IDE
- MySQL
